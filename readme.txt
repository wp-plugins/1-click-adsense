== Description ==

1-Click-AdSense inserts Google Adsense Ads automatically into all pages, posts and archives of your website. Optimized positions and sizes are used automatically to get the best compromize between minimum effect on weblayout and maximum income. 

Just activate and do 2 settings.....

== Installation ==

1. Upload `1-click-adsense` folder to the `/wp-content/plugins/` directory .
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Settings can be configured in the 'Options' area of your blog.
